# Graduation-Thesis
Agent-Based Macroeconomic Model with Basic Income

（Takashima & Yagi, 2023 再現モデル）

1. モデル概要

本モデルは、Takashima & Yagi (2023)
“Multilayered Emergent Phenomena Caused by Basic Income and Labor Supply on the Wider Economic System”
で提案された エージェントベース・マクロ経済モデル（ACE） を、
Python（Google Colab）で再現・拡張することを目的とする。

本モデルの特徴は以下である。

家計・企業・政府・銀行・設備メーカーの 完全な資金循環

家計の 労働意欲（work motivation） が内生的に変化

労働意欲 → 生産 → 価格 → 所得 → 労働意欲、という 多層的フィードバック

ベーシックインカム（BI）導入による マクロ創発現象 の分析

2. エージェント構成

本モデルには以下の5種類のエージェントが存在する。

Household（家計）
労働供給・消費・貯蓄・労働意欲決定

Firm（消費財企業）
生産・価格調整・賃金支払・設備投資

Government（政府）
税徴収・政府購入・補助金・BI給付

Bank（銀行）
預金管理・融資・利息支払

Equipment Maker（設備メーカー）
設備財生産・雇用（簡略化された企業）

3. 市場と財

消費財は I 種類 存在する

各企業は 2種類の消費財 を生産（シミュレーション中は固定）

同一財を複数企業が生産するため 競争市場 が成立

家計は 3種類の財のみを認知 し、それらに対して選好を持つ

家計の選好は Cobb–Douglas型効用関数

6. 実装上の重要ルール（Codex向け）

この README.md に書かれた仕様を 唯一の正 とする

Step0〜Step2 は 初期フェーズ として明確に分離

在庫は Step2 でのみ増加する

Step3 では 在庫減少と売上確定のみ

二重カウントは禁止

各Step後に 会計整合性チェック を行う

7. 実験パラメータ

期間数：360

家計：500

企業：60

財種類：10

BI 比率：0 / 20% / 40%

労働意欲パラメータ：ε = −0.02, −0.1

8. 目的

本リポジトリは以下を目的とする。

論文モデルの忠実な再現

BI・労働意欲・供給制約の相互作用分析

卒業論文・研究用シミュレーション基盤の構築
